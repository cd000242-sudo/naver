💡 예약 발행 사용 방법

1단계: "스마트 자동 발행" 탭에서 글을 생성합니다.

2단계: "발행 모드"를 "예약 발행"으로 선택합니다.

3단계: 원하는 날짜와 시간을 설정하고 "확인" 버튼을 클릭합니다.

4단계: "반자동 발행" 또는 "전자동 발행" 버튼을 클릭합니다.

5단계: 앱을 켜둔 상태로 대기하면 설정한 시간에 자동으로 발행됩니다!

⚠️ 중요 안내

• 예약 발행이 작동하려면 앱이 실행 중이어야 합니다.

• 앱을 최소화해도 괜찮지만, 완전히 종료하면 예약 발행이 실행되지 않습니다.

• "환경설정"에서 "계정 정보 기억하기"를 체크해야 자동 로그인됩니다.

API 키는 어디서 발급받나요?
환경설정(⚙️)을 열면 각 API 옆에 "발급 가이드" 버튼이 있습니다. 클릭하면 해당 서비스의 API 키 발급 페이지로 이동합니다.
무료로 사용할 수 있는 이미지 소스는?
Stability AI는 초고화질 실사 생성을 위한 유료 엔진입니다. DALL-E 3는 OpenAI의 엔진을 사용하며, Prodia는 가성비 좋은 옵션입니다. 나노 바나나 프로/Imagen 4는 Gemini API 키를 사용하며 Google 정책/계정 설정에 따라 과금될 수 있습니다.
예약 발행은 잘 작동하나요?
✅ 네, 예약 발행은 정상적으로 잘 작동합니다!

예약 발행을 사용하려면:
• 앱이 실행 중인 상태를 유지해주세요 (최소화 OK)
• 환경설정에서 "계정 정보 기억하기"를 체크해주세요
• 스케줄 관리 탭에서 예약된 목록을 확인할 수 있습니다
풀오토 발행은 실제 이미지 못 넣나요?
⚠️ 네, 현재 풀오토 발행에서는 AI 생성 이미지만 지원됩니다.

이유: 실제 이미지를 수집하려면 네이버/구글에서 이미지를 가져와야 하는데, 마음에 안 드시는 이미지가 들어갈 확률이 높습니다. 만족도가 떨어지기 때문에 해당 기능은 제외되었습니다.

🎯 콘텐츠 모드 가이드

🔍 SEO 모드란?

검색 상위노출에 최적화된 글을 생성합니다.
네이버 검색에서 블로그 글이 상위에 노출되도록 키워드 배치와 글 구조를 최적화합니다.

✨ 특징
  • 키워드 자연스럽게 5~7회 배치
  • 3,500~4,500자 긴 글 작성
  • 체류시간 증가를 위한 구조화된 콘텐츠
  • H2/H3 소제목 최적화
💡 추천: 정보성 글, 리뷰, 꿀팁 등 검색 유입을 원할 때

🏠 홈판 모드란?

네이버 메인·추천 영역 노출을 목표로 하는 글을 생성합니다.
트렌디하고 클릭을 유도하는 제목과 서두로 구성됩니다.

✨ 특징
  • 클릭 유도형 제목 (호기심 자극)
  • 짧고 임팩트 있는 서두
  • 트렌디한 문체 사용
  • 감성적/개인적 경험 톤
💡 추천: 일상 공유, 후기, 감성글 등 소셜 확산을 원할 때

🛒 쇼핑 커넥트 모드란?

네이버 쇼핑 커넥트 활동을 위한 구매 전환 글을 생성합니다.
모든 이미지에 제휴 링크가 자동 삽입되며, 인용구 4 스타일이 적용됩니다.

⚠️ 사용 가능 카테고리

아래 카테고리를 선택할 때만 이 모드가 활성화됩니다:

🛒 상품리뷰 🚗 자동차 💻 IT·컴퓨터 👗 패션·미용 🏥 건강·의학
✨ 특징
  • 모든 이미지에 제휴 링크 자동 삽입
  • 인용구 4 스타일 (밑줄 강조) 적용
  • 고정 문구: "이 포스팅은 네이버 쇼핑 커넥트 활동의 일환..."
  • 후킹 이미지 영상 변환 옵션 (VEO)

✏️ 사용자정의 모드란?

직접 프롬프트를 입력하여 원하는 형태의 글을 생성합니다.
고급 사용자를 위한 모드로, AI에게 세부 지시사항을 전달할 수 있습니다.

✨ 활용 예시
  • "마지막에 '인플루언서 맞팬 환영해요!' 넣어줘"
  • "특정 제품의 장점을 더 강조해줘"
  • "5,000자 이상으로 작성해줘"
  • "질문 형식의 서두로 시작해줘"
💡 추천: 특별한 요구사항이 있거나 실험적인 글을 작성할 때
