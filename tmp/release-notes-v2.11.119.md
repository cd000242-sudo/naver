## v2.11.119

### DropShot 로그인 창 반복 표시 수정

- 로그인 인식이 끝나면 보이는 Chrome 로그인 창을 완전히 닫도록 수정했습니다.
- 로그인 직후 불필요한 브라우저를 다시 열지 않고, 실제 이미지 생성 시점에만 숨김 세션을 준비합니다.
- persistent Chrome이 함께 만드는 중복 `about:blank` 탭을 정리합니다.
- 로그인 창 종료 직후 프로필 잠금이 잠깐 남는 경우에도 숨김 세션에서만 제한 재시도하며 새 창을 띄우지 않습니다.

### 공정위 문구 원문 보호

- 쇼핑커넥트 모드 전환이 사용자의 공정위 프리셋과 직접 입력 문구를 기본 문구로 덮어쓰던 동작을 제거했습니다.
- 모드 전환은 공정위 문구의 ON/OFF만 결정하며 저장된 원문은 그대로 유지합니다.
- AI가 본문에 `[광고]` 또는 제휴 고지 문구를 임의로 생성·요약·변형·중복 삽입하지 않도록 작성 규칙을 수정했습니다.

### 회귀 방지

- DropShot 창 종료·빈 탭 정리·숨김 인계·프로필 잠금 재시도와 공정위 원문 보존 전용 테스트를 추가했습니다.
- 전체 테스트 5,487개, TypeScript 빌드와 린트를 통과했습니다.
