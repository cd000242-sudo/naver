## v2.11.105

### 에이전트 구독 상태 및 생성 파이프라인 안정화

- Claude Code의 설치, 구조화된 로그인 출처, 활성 구독 상태를 분리해 확인하도록 개선했습니다.
- 만료되거나 비활성화된 구독은 실제 권한 점검으로 감지하며, 모든 에이전트 발행 모드에서 작업 시작 전에 명확하게 안내합니다.
- Claude Code 구독 모드가 API 키, 게이트웨이 또는 클라우드 제공자 과금 경로로 우회하지 않도록 실행 환경을 격리했습니다.
- Codex 에이전트도 ChatGPT 구독 로그인만 허용하고 API 키 과금 로그인은 발행 전에 차단합니다.
- Codex/Claude 자식 프로세스에는 실행·프로필 경로만 전달하고 네이버 비밀번호, API 키, GitHub 토큰 등 앱 비밀 환경변수는 기본 차단합니다.
- 한글 Windows 사용자 경로에서도 Codex/Claude CLI의 실제 실행 파일을 안전하게 찾아 실행하도록 개선했습니다.
- 구독 만료와 사용량 제한 오류에 대한 불필요한 콘텐츠 재시도를 중단했습니다.
- Gemini, OpenAI, Claude의 가성비·균형·프리미엄 모델 정책과 비용·할당량 계산을 최신 구성으로 정리했습니다.
- OpenAI 웹 검색을 실제 지원되는 GPT-5.6 Responses API 경로로 교체했습니다.
- 스타일 미리보기 이미지 IPC 중복 등록을 제거해 선택한 나노바나나 모델이 다른 엔진으로 바뀌지 않도록 수정했습니다.
- 종료된 무료 Gemini 이미지 모델 설정은 모든 재시도·폴백보다 먼저 차단하며, 네트워크 요청 없이 비용 확인 후 현재 모델을 다시 선택하도록 안내합니다.
- 화면에서 명시 선택한 이미지 모델을 배치 전체와 재시도까지 그대로 전달해 저장된 이전 설정이 비용·품질을 바꾸지 못하도록 고정했습니다.
- 이미지 결제·모델 정책 오류가 발생하면 진행 중 세션과 대기 작업을 즉시 중단해 추가 요청과 불필요한 재시도를 막았습니다.
- 렌더러 번들에 누락되던 Gemini 모델 정규화 함수를 포함해 앱 시작 오류를 방지했습니다.
- GitHub 릴리스 푸시 과정에서 토큰이 프로세스 인수나 로그에 노출되지 않도록 강화했습니다.
- 패키지 검증 종료 시 Electron 자식 프로세스와 임시 프로필을 확실히 정리하고, 중간 실패를 성공 코드로 오인하지 않도록 릴리스 파이프라인을 강화했습니다.
- 인증 직후 도착한 최신 공지가 시작 시 보관된 이전 공지에 덮이지 않도록 실시간 수신과 캐시 복구 순서를 고정했습니다.
- 콘텐츠 품질 보정, 이미지 엔진 선택, 로그 개인정보 제거 흐름의 회귀 테스트를 보강했습니다.

### 검증

- 단위·통합 테스트 4,468개, TypeScript/렌더러 빌드, IPC 계약 검사, 런타임 스모크 테스트, Electron UI E2E 11개를 통과했습니다.
