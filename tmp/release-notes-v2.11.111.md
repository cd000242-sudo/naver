## v2.11.111 — 리더스 나노바나나프로 무제한 로그인 안정화

- 저장된 Dropshot 로그인 계정을 자동으로 인식하며, 이미 로그인된 경우 새 브라우저 창을 열지 않습니다.
- 로그인 성공 직후 창을 닫고 헤드리스 브라우저를 다시 실행하던 구조를 제거했습니다.
- OAuth 로그인을 완료한 브라우저 컨텍스트를 그대로 재사용하고 최소화하여 창 열림·닫힘 반복과 프로필 잠금 경쟁을 차단했습니다.
- 로그인 토큰과 무제한·0비용 생성 준비 상태를 분리했습니다. 화면 로딩 지연을 로그아웃으로 오판하지 않습니다.
- 이미지 관리, 이미지 스튜디오, 풀오토 이미지 설정의 로그인 상태를 동기화하고 자동 인식 성공 알림을 추가했습니다.
- 실제 생성 직전 무제한 모드와 0비용 버튼 확인은 그대로 강제하여 Dropshot 코인 오사용을 방지합니다.
- 외부 AI/API 호출 없이 전체 5,441개 회귀 테스트, 린트, TypeScript/Electron 빌드를 통과했습니다.
