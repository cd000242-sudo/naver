## v2.11.118

### Claude Code 배포 앱 활성 복원

- 배포 앱에서도 **에이전트 모드(Claude Code 구독)**를 선택할 수 있도록 복원했습니다.
- Claude Code 설치, 상태 확인, 로그인, 계정 전환, 글생성 기능을 다시 활성화했습니다.
- 기존에 저장한 `agent-claude` 설정이 Gemini 또는 Claude API 키 모델로 자동 변경되지 않도록 수정했습니다.
- 로그인 상태를 자동 인식하면 **Claude Code 로그인 자동 인식 완료** 알림을 표시합니다.
- 설치되지 않았거나 로그인이 필요한 경우 차단 문구 대신 실제 설치·로그인 버튼과 원인을 표시합니다.

### 회귀 방지

- 패키지 여부와 무관하게 Claude Code가 활성 상태인지 검증하는 전용 회귀 테스트를 추가했습니다.
- 전체 테스트 5,480개와 TypeScript 빌드를 통과했습니다.
