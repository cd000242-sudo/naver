## v2.11.38

### 발행 안정화 긴급 패치

- 발행 막바지 88~95% 구간에서 `브라우저 세션 복구 중` 이후 재로그인되거나 실패하던 문제를 안정화했습니다.
- 네이버 글쓰기 에디터가 늦게 열릴 때 브라우저를 바로 닫지 않고, 같은 창에서 다시 확인하도록 개선했습니다.
- 실제 브라우저 세션이 끊긴 경우에는 전체 자동화를 취소하지 않고 해당 계정 세션만 정리하도록 수정했습니다.
- 반자동 발행에서 이미지 관리 탭이 비어 있어도, 사용자가 원한 경우에는 이미지 없이 본문만 발행할 수 있도록 정리했습니다.
- 관련 회귀 테스트를 추가해 같은 문제가 다시 생기면 테스트 단계에서 잡히도록 했습니다.

### 검증

- 전체 테스트 통과
- 빌드 통과
- IPC 채널 검증 통과
