## v2.11.102 안정화 릴리스

### 품질 점수와 자동 발행
- 90점은 품질 목표로 유지하되 자동 발행의 절대 합격선으로 사용하지 않도록 수정했습니다.
- 종합 80점, 모드 75점, 안전성 70점 이상인 안전한 원고는 90점 미만이어도 근접 통과로 발행을 계속합니다.
- 홈판·쇼핑 글은 사람다움 70점 기준을 추가로 유지하며, 정책 위반·사실 오류·최소 분량 미달은 기존처럼 차단합니다.
- 쇼핑커넥트의 별도 점수 검사도 80점 이상이면 근접 통과하도록 동일한 기준을 적용했습니다.

### 쇼핑 상품 수집 정확성
- 네이버 상품 페이지에서 배송비·적립금·할부 금액을 판매가로 오인하거나 여러 숫자를 이어 붙이는 문제를 차단했습니다.
- 판매가가 여러 개로 모호하면 임의로 가장 큰 숫자를 선택하지 않고 검증 가능한 가격만 사용합니다.
- JSON-LD와 실제 상품 갤러리를 함께 분석해 대표 이미지와 추가 이미지를 원래 순서대로 수집합니다.
- 구매후기 이미지는 공식 상품 이미지와 분리해 원본 상품 갤러리에 섞이지 않도록 수정했습니다.
- 브랜드스토어 수집은 유지보수 중인 Provider 경로를 우선 사용하고 장시간 대기하는 레거시 이중 브라우저 실행을 기본적으로 생략합니다.

### 콘텐츠 정책과 공지
- 실제 수집 근거가 있는 쇼핑 글이 일반적인 정책 문구 때문에 잘못 차단되는 경로를 보강했습니다.
- 관리자 화면에서 저장한 최신 공지가 인증 직후 서버 캐시에 덮어쓰이지 않고 사용자에게 바로 표시되도록 수정했습니다.
- 공지 HTML 표시와 다시 보기 상태를 동일한 런타임 경로로 통일했습니다.

### 회귀 방지
- 품질 근접 통과, 가격 정규화, 브랜드스토어 DOM·JSON-LD, 공식 이미지 갤러리, 콘텐츠 근거, 공지 표시 회귀 테스트를 추가했습니다.
- 전체 테스트, 린트, TypeScript 빌드, Electron UI E2E, IPC 계약, 설치본 클린 프로필 스모크를 릴리스 게이트에서 검증합니다.
