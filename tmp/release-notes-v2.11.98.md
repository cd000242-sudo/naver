## Better Life Naver 2.11.98

### 이번 업데이트 핵심
- 연속발행에서 Flow 이미지 생성이 끝난 뒤 네이버 로그인과 발행 단계로 넘어가지 않던 문제를 수정했습니다.
- 이전 풀오토 작업의 취소 상태가 연속발행에 섞이지 않도록 진행 모달과 실행 상태를 분리했습니다.
- 실제 네이버 발행 요청 전 오류와 요청 후 결과 미확정 상태를 구분해 안전한 경우에만 자동 재시도합니다.
- 매출, 비용, 순이익, 채널별 성과와 월 목표 달성 여부를 기록하는 수익 운영 대시보드를 추가했습니다.
- 이미지 설정 버튼이 필요한 화면에서 안정적으로 표시되도록 UI 연결을 보강했습니다.

### 안정성 개선
- 연속발행 상태를 내부 상태와 브라우저 전역 상태에 함께 동기화해 다음 글에서 설정이 풀리는 회귀를 차단했습니다.
- 이전글 자동 연결 데이터에 문제가 있어도 네이버 로그인과 본문 발행은 계속 진행하도록 보호했습니다.
- Flow 이미지 완료 후 네이버 발행 엔진으로 작업이 전달되는 시점을 별도로 기록해 중복 발행 위험을 줄였습니다.
- 관련 회귀 테스트와 전체 테스트, TypeScript 빌드, ESLint 검증을 통과했습니다.

기존 사용자는 앱 업데이트 후 그대로 사용하면 됩니다.
