## v2.11.31

네이버 글쓰기 화면에서 제목 입력 영역을 찾지 못해 통합 자동화가 중단되던 문제를 보강했습니다.

- 네이버 에디터 제목 영역이 늦게 뜨거나 구조가 조금 달라져도 여러 방식으로 다시 찾도록 개선
- 제목 입력 실패 시 어떤 화면/프레임/에디터 상태에서 막혔는지 로그가 더 자세히 나오도록 개선
- 본문과 이미지 삽입 전 커서가 제목 영역에 남아있는지 확인하는 로직 안정화
- 같은 문제가 다시 생기지 않도록 제목 셀렉터 fallback 테스트 추가

이번 업데이트는 발행 중 갑자기 `documentTitle` 셀렉터 오류가 뜨는 케이스를 줄이는 안정화 패치입니다.
