## v2.11.103

### 런타임 회귀 방지

- 쇼핑 이미지 정책 함수가 개발 테스트에서는 통과하고 실제 설치 앱 번들에서 누락되던 문제를 수정했습니다.
- 필수 렌더러 함수가 최종 번들에 없으면 빌드 단계에서 즉시 실패하도록 검증을 강화했습니다.
- 반자동 편집 글의 순서 보존과 IPC 전달을 실제 Electron 화면에서 검증하는 진단형 E2E를 보강했습니다.

### 개선 사항

- 쇼핑커넥트 대표 이미지는 원본을 썸네일로 유지하고, AI 소제목 이미지는 대표 이미지를 레퍼런스로 사용하도록 통일했습니다.
- 수집 이미지 모드에서는 대표 이미지 다음인 2번 이미지부터 소제목에 배치되도록 모든 발행 경로를 맞췄습니다.
- 나노바나나2, OpenAI 이미지, 리더스 나노바나나프로 무제한의 레퍼런스 이미지 전달을 강화했습니다.
- 드롭샷 로그인 확인 창은 완료 후 닫고, 실제 이미지 생성은 숨김 브라우저 세션으로 이어지도록 수정했습니다.
- 에이전트 생성 글의 `[출처: ...]` 같은 내부 표기만 제거하고 자연스러운 근거 문장은 유지하도록 보강했습니다.
- 풀오토, 연속 발행, 다중 계정, 반자동 발행 경로에 동일한 이미지 및 본문 정리 규칙을 적용했습니다.
- 기존 설치의 쇼핑 AI 엔진 설정을 자동 승계해 업데이트 후 선택 엔진이 임의로 바뀌지 않도록 했습니다.

### 안정성

- 외부 레퍼런스 이미지 로더에 사설망 접근, DNS 재바인딩, 위험한 리다이렉트 차단을 추가했습니다.
- 로컬 레퍼런스 이미지는 앱 관리 폴더 안의 실제 이미지 파일만 허용하고, UNC 경로와 과대 데이터 이미지를 차단했습니다.
- 대표 이미지가 없는 쇼핑 AI 발행은 일반 SEO 이미지 생성으로 우회하지 않고 명확히 중단하도록 보강했습니다.
- 쇼핑 이미지 순서, 구버전 설정 승계, 드롭샷 숨김 세션, 출처 표기 제거에 대한 회귀 테스트를 추가했습니다.
