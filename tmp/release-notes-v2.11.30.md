## v2.11.30 업데이트

이번 버전은 자동 발행 마지막 단계와 이미지 배치 안정성을 집중 개선했습니다.

- 썸네일 텍스트가 모든 이미지에 들어가던 문제를 수정했습니다. 이제 대표 썸네일에만 제목 텍스트가 들어갑니다.
- 이미지 생성 중단 후 다른 이미지 엔진으로 다시 실행할 때, 이전 실패 이미지가 남아 배치되는 문제를 줄였습니다.
- 이전글 엮기와 해시태그가 본문 리치 붙여넣기 이후 누락되는 흐름을 보강했습니다.
- CTA 위치 선택 기능을 다시 사용할 수 있게 정리했습니다.
- 풀오토/반자동 이미지 미리보기와 발행 메타데이터 전달 안정성을 개선했습니다.

업데이트 후에도 기존 설정과 API 키는 그대로 유지됩니다.
