## v2.11.120

### OpenAI 글 생성 실패 진단·재호출 방지

- `gpt-5.6-sol`의 장문 생성 제한을 240초, `gpt-5.6-terra`는 180초로 조정해 기존 90초 조기 종료를 방지했습니다.
- GPT-5.6 추론 강도를 공식 권장 균형값인 `medium`으로 조정해 지연·비용·출력 여유를 개선했습니다.
- 타임아웃, 빈 응답, 인증, 결제, 속도 제한, 서버 오류를 구분해 실제 원인과 요청 ID를 안내합니다.
- 빈 200 응답에서도 사용량을 먼저 기록하고, `finish_reason`과 출력·추론 토큰 정보를 오류에 포함합니다.
- OpenAI 실패는 생성 파이프라인 바깥 단계에서 자동 재호출하지 않으며, 유료 모델 자동 폴백도 수행하지 않습니다.
- 실패 메시지에 실제 요청 횟수와 `자동 재호출 0회`를 표시하고 과금 가능성을 사실대로 안내합니다.

### 보안·회귀 방지

- 오류와 진단 로그에서 API 키, Bearer 토큰, JWT, URL 자격 증명, 비밀 쿼리 값을 제거합니다.
- HTTP 상태 코드를 우선해 403·429·5xx가 네트워크 장애로 오인되지 않도록 수정했습니다.
- 비용 불변식, 타임아웃 정책, 모델 설정, 오류 분류, 바깥 재호출 차단 회귀 테스트를 추가했습니다.
