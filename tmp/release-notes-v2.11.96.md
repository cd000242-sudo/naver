## Better Life Naver v2.11.96

### 주요 개선

- 쇼핑커넥트 글을 근거 유형에 따라 작성하도록 개선했습니다. 사용자가 입력한 직접 사용 메모가 없으면 AI가 사용 경험을 꾸며내지 않습니다.
- 과장된 긴급성, 가짜 만족도·인기 표현, 반복 구매 링크를 차단하고 대표 상품 이미지와 하단 안내 영역에만 링크를 배치합니다.
- 반자동 편집 글의 문단·소제목·목록·표 순서를 보존하는 발행 경로와 임시·예약 발행 검증을 강화했습니다.
- 드롭샷 로그인 창, 브라우저 세션 복구, 흰 화면·조기 종료·타임아웃 처리와 이미지 작업 정리를 보강했습니다.
- 최근 게시물 기반 유사도 검사, 품질·발행 차단, 노출 모니터링 정책을 기본 설정과 함께 배포합니다.
- 사용자 PC에서 기본 정책과 프롬프트가 빠지지 않도록 패키징·릴리스 검증을 강화했습니다.

### 품질 검증

- ESLint, 전체 Vitest 회귀 테스트, TypeScript 및 렌더러 프로덕션 빌드를 통과한 경우에만 설치 파일을 생성합니다.
- Git 소스 푸시가 실패하면 GitHub 릴리스 업로드도 중단하여 설치 파일과 소스 버전의 불일치를 방지합니다.
