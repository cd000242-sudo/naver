## Better Life Naver v2.11.127

### 핵심 수정

- 내부 콘텐츠 품질·정책·노출 진단은 글 생성과 발행을 막지 않고 경고로 안내합니다.
- 과거 자동 노출 점검으로 남은 일시정지 상태를 안전하게 복구합니다.
- 발행 성공 후 내부 노출 추적 기록이 실패해도 다음 글 생성·발행을 전역 차단하지 않습니다.
- MCP 연결 실험 기능을 제거하고 기존 API 키·Codex·Claude Code 선택 구조로 정리했습니다.
- 쇼핑커넥트 리뷰 근거 수집과 후기형 글 프롬프트를 강화했습니다.
- 모든 이미지 생성 엔진이 글 전체 맥락과 소제목 의미를 함께 반영하도록 프롬프트를 개선했습니다.
- 생성 글 목록·불러오기 흐름과 공지 UI를 정리했습니다.

### 안정성

- 선택하지 않은 글·이미지 엔진으로 자동 폴백하지 않는 비용 불변식을 유지합니다.
- 빈 원고, 사용자 취소, 인증·네트워크 실패, 잘못된 예약 정보 등 실제 실행 불가능 상태는 정확한 원인과 함께 중단합니다.
