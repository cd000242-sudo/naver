# Better Life Naver v2.11.124

## 드롭샷 로그인 자동 인식 및 창 숨김 수정

- 현재 드롭샷의 Auth.js 서버 세션을 기준으로 로그인 완료를 정확히 감지합니다.
- 로그인 완료 후 해당 Chrome 창을 즉시 최소화하고 같은 인증 세션을 이미지 생성에 재사용합니다.
- 로그인 직후 리다이렉트가 끝나기 전에도 완료된 세션을 인식합니다.
- AI Studio 탭을 다른 드롭샷 보조 탭보다 우선 선택합니다.
- OAuth 임시 `about:blank` 팝업을 잘못 닫지 않으며, 실제로 닫힌 팝업과 종료된 메인 창을 구분합니다.
- 서버 세션 확인 실패 시 남아 있는 구형 토큰을 로그인으로 오인하지 않습니다.
- 세션 검사에 브라우저·앱 양쪽 타임아웃과 정확한 도메인 재검증을 적용했습니다.
- 세션·토큰·사용자 식별 정보는 앱 프로세스나 로그로 전달하지 않습니다.

## 검증

- 실제 드롭샷 전용 프로필: 보드 진입, 세션 API 200, 로그인 판정 성공 확인
- 드롭샷 관련 전체 회귀 테스트 통과
- TypeScript 빌드 및 ESLint 오류 0건 확인
