## v2.11.126

### MCP 연결 화면 단순화

- 글·이미지의 에이전트/API 선택을 중복 표시하던 연결 화면을 MCP 전용 선택 화면으로 정리했습니다.
- MCP를 선택하지 않으면 기존 AI 텍스트 엔진과 풀오토·이미지 관리 탭의 설정을 그대로 사용합니다.
- MCP 글 생성은 기존 API 경로에서 완성된 최종 프롬프트를 내용 변경 없이 전달합니다.
- MCP 실패 시 다른 엔진이나 유료 API로 자동 폴백하지 않습니다.

### 쇼핑커넥트 이미지 엔진 회귀 수정

- 이전 연결 화면에 남은 비-MCP 이미지 경로가 현재 이미지 탭의 선택을 덮어쓰지 못하도록 수정했습니다.
- 기존 두 이미지 설정이 모두 Dropshot 선택과 일치하는 경우에만 쇼핑커넥트의 오래된 엔진 값을 리더스 나노바나나프로 무제한(Dropshot)으로 안전하게 한 번 이관합니다.
- 현재 이미지 설정이 더 최신이면 저장값을 건드리지 않고 오래된 중복 경로만 제거합니다.
- 리더스 무제한은 Dropshot 내부 Nano Banana Pro 모델을 사용하며 Gemini API 키 과금 경로가 아님을 화면에 명시했습니다.

### 안정성

- 자동화 파이프라인 시작 전에 기존 중복 경로 정리가 완료되도록 초기화 순서를 고정했습니다.
- MCP만 별도 실행 경로를 덮어쓸 수 있도록 글·이미지 실행 경계를 고정했습니다.
- 관련 회귀 테스트와 런타임 소스 지문을 갱신했습니다.
