## Better Life Naver 2.11.99

### 발행 파이프라인 안정화
- 반자동, 풀오토, 연속발행, 다중계정, 앱 예약발행, 스마트 예약발행에서 최종 제목과 본문을 기준으로 콘텐츠 정책 상태를 다시 맞추도록 개선했습니다.
- 이전 생성 단계의 제목이나 본문이 남아 정상 글을 오탐 차단하던 문제를 방지하면서, 실제 제목·본문 불일치는 계속 차단합니다.
- 임시저장과 예약발행은 발행 간격 제한에서 제외하되 콘텐츠 안전성과 작업 상태 검사는 유지합니다.
- 스마트 예약발행이 키워드만 본문으로 넘기지 않고 완성된 초안을 생성한 뒤 발행하도록 보강했습니다.

### 이미지 및 콘텐츠 품질 개선
- 수집 이미지는 중복을 제거하고, AI 이미지 모드에서는 대표 이미지를 레퍼런스로 사용해 새 이미지를 생성하도록 경로를 정리했습니다.
- Dropshot 이미지 생성의 레퍼런스 전달과 실패 시 대체 경로를 안정화했습니다.
- 출처로 확인되지 않은 수치·가격·후기성 사실을 자동으로 검출하고 복구하거나 차단하도록 품질 게이트를 강화했습니다.
- 일반 정보 글과 업체·제휴 글의 사실 검증 기준을 구분해 정상 정보 글이 불필요하게 막히지 않도록 했습니다.

### 검증
- 콘텐츠 정책, 모든 발행 모드, 쇼핑 이미지 레퍼런스, 이미지 중복 제거, 품질 게이트 회귀 테스트를 추가했습니다.
- 전체 테스트, TypeScript 빌드, ESLint, IPC 연결 검사와 릴리스 게이트를 통과한 설치본입니다.
