## v2.11.123

### 리더스 나노바나나 로그인 안정화

- 로그인 성공 직후 전용 자동화 창과 OAuth 보조 창을 모두 최소화하고, 인증된 동일 세션을 이미지 생성에 재사용합니다.
- 로그인 창을 닫았다가 다시 여는 과정에서 발생하던 `about:blank` 새 창 반복과 프로필 잠금 경쟁을 제거했습니다.
- 종료가 완료되지 않은 전용 브라우저가 있으면 새 창을 열지 않으며, 프로필 잠금 오류에서도 Chrome·Edge·Chromium을 연속 실행하지 않습니다.
- 로그인·세션 오류가 발생한 이미지 배치는 첫 실패에서 중단해 이미지 개수만큼 로그인 창이 반복되는 현상을 막았습니다.
- 늦게 도착한 과거 상태 확인 응답이 로그인 성공 UI를 다시 로그인 버튼으로 되돌리지 않도록 요청 순서를 보호했습니다.
- 로그인 성공 시 계정 인식 알림을 표시하고, 생성 중 상태 확인 응답 때문에 버튼이 계속 비활성화되는 문제를 수정했습니다.
