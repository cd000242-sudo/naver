## v2.11.44

### 발행 안정화 긴급 수정

- 네이버 글쓰기 마지막 단계에서 본문이 정상 작성됐는데도 해시태그 검증 오류로 발행이 중단되던 문제를 수정했습니다.
- 발행 직전 검증기가 네이버 에디터의 `사진 편집`, `문서 너비`, `AI 사용 설정` 같은 UI 문구를 본문으로 잘못 읽는 경우를 분리했습니다.
- 해시태그/이전글 엮기 마무리 검증이 불안정해도 같은 글을 새 창에 다시 작성하지 않도록 안정화했습니다.
- 이전글 카드 또는 CTA 카드 뒤에는 공백을 확보한 뒤 해시태그 입력을 이어가도록 회귀 테스트를 강화했습니다.
