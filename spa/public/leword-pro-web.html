추가 기능

각 기능은 시드 입력 없이 서버가 실시간 소스와 카테고리를 감지해 자동 발굴합니다.
자동 발굴 모드 실시간 소스와 카테고리를 서버가 감지해서 후보를 수집하고, 검색량·문서수·니즈 의도를 교차검증합니다.
Pro 트래픽 폭발 키워드 헌터가 실시간 이슈와 시즌 신호를 자동으로 감지합니다.

도구 선택 대기

서브탭에서 기능을 선택하고 실행하면 진행률 모달이 뜬 뒤 결과만 정리해서 표시합니다.

유튜브 황금키워드 및 쇼츠분석

유튜브 전용 사이드 화면입니다. 추가 기능 서브탭과 분리해 급상승 영상/쇼츠 신호만 분석합니다.
유튜브 자동 분석 YouTube Data API 키가 있으면 급상승 영상과 쇼츠 후보를 수집하고, 네이버 실측 키와 함께 교차검증합니다.

환경설정

Electron 앱처럼 각 사용자 API 키를 입력합니다. 키는 이 브라우저 로컬에만 저장되고, 서버 공용 설정으로 저장하지 않으며 실행 요청에만 전달됩니다.

키워드 API 설정

네이버 API는 문서수/연관어, 네이버 검색광고 API는 PC/모바일 실측 검색량, 유튜브 API는 유튜브 황금키워드에 사용합니다.

Claude/Manus/OpenAI 키가 있으면 서버 job이 키워드 니즈 추론, 후보 의도 분류, 부적합 키워드 제거에 활용할 수 있게 요청 헤더로 전달합니다.
키 값은 서버에 저장하지 않습니다. 저장 후 비밀키 입력칸은 비우고, 실행할 때 브라우저 로컬 값만 요청 헤더로 전달합니다.

관리자 AI 작업자 설정

Codex와 Claude Code는 관리자 전용 서버 작업자입니다. 관리자 계정으로 Pro 로그인한 뒤 사용할 작업자를 선택하세요.

관리자 설정 잠김 관리자 로그인 후 비밀번호를 입력하면 Codex/Claude Code 작업자 설정이 열립니다.
일반 사용자는 볼 수 없고, 관리자 세션에서 실행하는 job 요청에만 작업자 선택 메타가 전달됩니다.
관리자 계정으로 로그인하면 Codex 또는 Claude Code 작업자를 선택할 수 있습니다.

사이트 콘텐츠 관리

관리자만 제품정보, 무료 챗봇, 구매 문구, 신규 제품 초안을 서버에 저장합니다. 저장 데이터는 /v1/public/site-content에서 공개 조회할 수 있습니다.

관리자 콘텐츠 잠김 관리자 로그인 후 비밀번호 확인을 완료하면 사이트 콘텐츠 편집기가 열립니다.
