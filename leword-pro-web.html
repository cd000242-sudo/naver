추가 기능

각 기능은 시드 입력 없이 서버가 실시간 소스와 카테고리를 감지해 자동 발굴합니다.
자동 발굴 모드 실시간 소스와 카테고리를 서버가 감지해서 후보를 수집하고, 검색량·문서수·니즈 의도를 교차검증합니다.
Pro 트래픽 폭발 키워드 헌터가 실시간 이슈와 시즌 신호를 자동으로 감지합니다.

환경설정

Electron 앱처럼 각 사용자 API 키를 입력합니다. 키는 이 브라우저 로컬에만 저장되고, 서버 공용 설정으로 저장하지 않으며 실행 요청에만 전달됩니다.

키워드 API 설정

네이버 API는 문서수/연관어, 네이버 검색광고 API는 PC/모바일 실측 검색량, 유튜브 API는 유튜브 황금키워드에 사용합니다.

Claude/Manus/OpenAI 키가 있으면 서버 job이 키워드 니즈 추론, 후보 의도 분류, 부적합 키워드 제거에 활용할 수 있게 요청 헤더로 전달합니다.
