# Better Life naver - 네이버 블로그 자동화

더 나은 삶을 위한 AI 기반 블로그 자동화 도구  
콘텐츠 생성부터 자동 포스팅까지, 네이버 블로그 관리를 한 번에 처리하는 Electron 데스크톱 애플리케이션입니다.

## 주요 기능

### 🤖 AI 콘텐츠 생성
- **구조화 콘텐츠 자동 생성**: RSS 피드, URL, 키워드, 초안 텍스트를 입력하면 AI가 제목, 소제목, 본문, 해시태그를 자동으로 생성합니다.
- **다중 AI 엔진 지원**: Gemini, Claude, OpenAI 중 선택 가능
- **소제목 기반 구조화**: 본문을 논리적인 소제목 단위로 자동 분할하여 가독성 높은 포스팅 생성

### 🎨 이미지 자동 생성 및 관리
- **DALL·E 3 이미지 생성**: 소제목별로 AI 이미지를 자동 생성
- **Pexels 이미지 라이브러리**: 키워드 기반으로 무료 이미지 자동 수집
- **이미지 라이브러리**: RSS/URL 크롤링 시 수집된 이미지를 카테고리별로 관리
- **이미지 자동 배치**: 소제목 아래에 이미지를 자동으로 삽입

### 📝 스마트 포스팅
- **글만 발행하기**: 이미 생성된 콘텐츠를 수정 없이 바로 발행
- **자동 생성만 실행**: 콘텐츠만 생성하고 발행은 하지 않음
- **발행 모드**: 임시저장, 즉시발행, 예약발행 지원
- **CTA 링크**: 포스팅 하단에 자동으로 링크 삽입

### 📊 품질 분석
- **AI 탐지 위험도**: 생성된 콘텐츠의 AI 탐지 가능성 분석
- **법적 위험도**: 저작권 및 법적 문제 가능성 체크
- **SEO 점수**: 검색 엔진 최적화 점수 제공
- **일일 발행 제한**: 하루 권장 발행 횟수 관리

## 플랫폼 지원

- **Windows**: ✅ 완전 지원 (Windows 10/11)
- **macOS**: ✅ 지원 (Intel/Apple Silicon 모두 지원)
- **Linux**: ✅ 지원 (Ubuntu, CentOS 등)

Electron 기반으로 모든 주요 데스크톱 플랫폼에서 실행 가능합니다.

## 준비 사항

- **운영체제**: Windows 10+, macOS 10.15+, Ubuntu 18.04+
- **Node.js 18 이상**
- **npm** 또는 다른 Node 패키지 매니저
- **네이버 계정** (아이디 / 비밀번호)
- **API 키** (선택 사항):
  - Gemini API Key (Google AI Studio)
  - OpenAI API Key (DALL·E 3 이미지 생성용)
  - Pexels API Key (이미지 라이브러리용)
  - Claude API Key (Anthropic, 선택 사항)
- **저장 공간**: Puppeteer가 Chromium을 다운로드하므로 수백 MB 필요

## 설치

```bash
# 저장소 클론 (또는 압축 해제)
cd "리더 네이버 자동화"

# 의존성 설치
npm install
```

> `puppeteer`가 설치되면서 Chromium이 자동으로 다운로드됩니다. 이미 설치된 Chromium/Chrome을 사용하려면 [`PUPPETEER_EXECUTABLE_PATH`](https://pptr.dev/guides/configuration) 환경변수를 설정하세요.

## 실행

### 데스크톱 앱 실행

```bash
npm start
```

- `npm start`는 TypeScript를 `dist/`로 빌드하고, Electron 앱을 실행합니다.
- 앱이 실행되면 UI에서 모든 작업을 수행할 수 있습니다.

> 개발 중 빠른 확인이 필요하면 `npm run build` 후 `npm run dev`로 바로 Electron을 실행할 수 있습니다. (코드 변경 시 다시 `npm run build` 필요)

### CLI 실행 (디버깅용)

```bash
npm run cli
```

환경 변수(`NAVER_ID`, `NAVER_PASSWORD`)나 `.env`에 계정 정보를 제공해야 합니다.

## 환경 설정

### 환경 변수 파일 (`.env`)

프로젝트 루트에 `.env` 파일을 생성하여 기본 설정을 저장할 수 있습니다:

```ini
# 네이버 계정 (선택)
NAVER_ID=your-id
NAVER_PASSWORD=your-password

# AI API 키 (선택)
GEMINI_API_KEY=your-google-ai-key
OPENAI_API_KEY=your-openai-key
PEXELS_API_KEY=your-pexels-key
CLAUDE_API_KEY=your-claude-key

# Gemini 모델 선택 (기본값: gemini-1.5-flash)
# GEMINI_MODEL=gemini-1.5-flash
```

> ⚠️ **보안 주의**: `.env` 파일은 버전 관리에서 제외되어야 합니다. 민감한 정보를 포함하므로 공유하지 마세요.

### 앱 내 환경 설정

앱 실행 후 상단의 **환경 설정** 섹션에서 API 키를 입력하고 저장할 수 있습니다. 입력한 키는 사용자 데이터 폴더에 저장됩니다.

## 사용 가이드

### 1. 기본 포스팅 (수동 입력)

1. **네이버 계정 입력**: 네이버 아이디와 비밀번호를 입력합니다.
2. **제목 및 본문 작성**: 제목과 본문을 직접 입력합니다.
3. **해시태그 추가** (선택): 해시태그를 입력합니다.
4. **발행 모드 선택**: 임시저장, 즉시발행, 예약발행 중 선택합니다.
5. **자동화 실행**: **자동화 실행** 버튼을 클릭합니다.

### 2. AI 구조화 콘텐츠 생성

#### 방법 1: RSS 피드 사용
1. **RSS URL 입력**: RSS 피드 URL을 입력합니다.
2. **AI 엔진 선택**: Gemini, Claude, OpenAI 중 선택합니다.
3. **자동 생성만 실행**: **자동 생성만 실행** 버튼을 클릭합니다.
4. **결과 확인**: 생성된 제목, 소제목, 본문을 확인하고 수정합니다.
5. **이미지 생성** (선택): 소제목별로 이미지를 생성합니다.
6. **발행**: **자동화 실행** 또는 **글만 발행하기** 버튼을 클릭합니다.

#### 방법 2: URL 크롤링
1. **URL 입력**: 크롤링할 웹페이지 URL을 입력합니다.
2. **AI 엔진 선택**: 원하는 AI 엔진을 선택합니다.
3. **자동 생성만 실행**: 콘텐츠를 생성합니다.
4. **결과 확인 및 수정**: 생성된 콘텐츠를 검토하고 필요시 수정합니다.
5. **발행**: 발행 버튼을 클릭합니다.

#### 방법 3: 키워드 기반 생성
1. **키워드 입력**: 콘텐츠와 관련된 키워드를 입력합니다.
2. **초안 텍스트** (선택): 참고할 초안 텍스트를 입력합니다.
3. **AI 엔진 선택**: 원하는 AI 엔진을 선택합니다.
4. **자동 생성만 실행**: 콘텐츠를 생성합니다.
5. **결과 확인 및 발행**: 생성된 콘텐츠를 확인하고 발행합니다.

### 3. 이미지 생성 및 관리

#### 이미지 자동 생성 (DALL·E 3)
1. **구조화 콘텐츠 생성**: 먼저 구조화 콘텐츠를 생성합니다.
2. **소제목 선택**: 이미지 탭에서 이미지를 생성할 소제목을 선택합니다 (최대 10개).
3. **프롬프트 수정** (선택): 각 소제목에 대한 이미지 프롬프트를 수정할 수 있습니다.
4. **이미지 생성**: **이미지 자동 생성** 버튼을 클릭합니다.
5. **결과 확인**: 생성된 이미지를 확인하고 필요시 다시 생성할 수 있습니다.

> ⚠️ **주의**: DALL·E 3는 콘텐츠 정책에 위배되는 프롬프트를 거부할 수 있습니다. 프롬프트를 수정하여 재시도하세요.

#### 이미지 라이브러리 (Pexels)
1. **이미지 탭 이동**: 이미지 관리 탭으로 이동합니다.
2. **키워드 입력**: 검색할 키워드를 입력합니다.
3. **이미지 수집**: **이미지 수집** 버튼을 클릭합니다.
4. **카테고리 관리**: 수집된 이미지를 카테고리별로 관리합니다.
5. **이미지 사용**: 발행 시 자동으로 사용되거나 수동으로 선택할 수 있습니다.

### 4. 글만 발행하기

이미 생성된 콘텐츠를 수정 없이 바로 발행하고 싶을 때 사용합니다.

1. **콘텐츠 확인**: 제목, 본문, 소제목이 모두 준비되었는지 확인합니다.
2. **글만 발행하기**: **글만 발행하기** 버튼을 클릭합니다.
3. **이미지 제외**: 이 모드에서는 이미지가 삽입되지 않고 텍스트만 발행됩니다.

> 💡 **팁**: 이미 생성된 콘텐츠를 수정한 후 다시 발행할 때 유용합니다.

### 5. 발행 모드

#### 임시저장 (기본값)
- 네이버 블로그에 임시저장으로 저장됩니다.
- 나중에 수동으로 발행할 수 있습니다.

#### 즉시발행
- 생성된 콘텐츠를 즉시 발행합니다.
- 발행 후 바로 블로그에 노출됩니다.

#### 예약발행
1. **예약발행 선택**: 발행 모드에서 "예약발행"을 선택합니다.
2. **날짜/시간 입력**: 예약할 날짜와 시간을 입력합니다.
3. **자동화 실행**: 지정된 시간에 자동으로 발행됩니다.

### 6. CTA 링크 추가

포스팅 하단에 링크를 자동으로 삽입할 수 있습니다.

1. **CTA 링크 입력**: 링크 URL을 입력합니다 (예: `https://example.com`).
2. **CTA 텍스트 입력**: 링크에 표시될 텍스트를 입력합니다 (예: "더 알아보기", "구매하기").
3. **자동화 실행**: 발행 시 자동으로 하단에 삽입됩니다.

### 7. 품질 분석

구조화 콘텐츠가 생성되면 자동으로 다음 지표가 표시됩니다:

- **AI 탐지**: AI로 생성된 콘텐츠로 탐지될 가능성
- **법적 위험**: 저작권 및 법적 문제 가능성
- **SEO 점수**: 검색 엔진 최적화 점수
- **일일 권장**: 하루 권장 발행 횟수 (설정에서 변경 가능)

## 타이핑 순서

자동화 실행 시 콘텐츠는 다음 순서로 입력됩니다:

1. **제목 입력**
2. **소제목 입력** (인용구 없이 일반 텍스트)
3. **Enter 키 입력**
4. **본문 영역 클릭**
5. **이미지 삽입** (생성된 이미지가 있는 경우)
6. **Enter 키 입력**
7. **정렬 설정** (가운데 정렬)
8. **본문 내용 입력**
9. **해시태그 입력**
10. **CTA 링크 삽입** (있는 경우)
11. **발행 모드에 따라 저장/발행**

> **글만 발행하기** 모드에서는 이미지 삽입이 건너뛰어지고 텍스트만 입력됩니다.

## 문제 해결

### 이미지가 생성되지 않을 때
- **API 키 확인**: OpenAI API Key가 올바르게 설정되었는지 확인하세요.
- **프롬프트 수정**: 콘텐츠 정책에 위배되는 프롬프트인지 확인하고 수정하세요.
- **재시도**: 이미지 생성 실패 시 프롬프트를 수정하여 다시 시도하세요.

### 본문 영역 클릭 실패
- **대기 시간**: 네이버 블로그 에디터가 완전히 로드될 때까지 기다려주세요.
- **수동 확인**: 브라우저 창에서 수동으로 본문 영역을 클릭한 후 다시 시도하세요.
- **로그 확인**: 로그 패널에서 상세한 오류 메시지를 확인하세요.

### AI 콘텐츠 생성 실패
- **API 키 확인**: 해당 AI 엔진의 API Key가 올바르게 설정되었는지 확인하세요.
- **인터넷 연결**: 인터넷 연결 상태를 확인하세요.
- **API 할당량**: API 사용량 제한에 도달했는지 확인하세요.

### 정렬 버튼을 찾을 수 없을 때
- **자동 처리**: 정렬 버튼을 찾지 못해도 본문 작성은 계속 진행됩니다.
- **수동 정렬**: 발행 후 네이버 블로그에서 수동으로 정렬을 설정할 수 있습니다.

## 주요 파일 설명

- **`src/main.ts`**: Electron 메인 프로세스. 창 생성, IPC 처리, 자동화 실행 관리.
- **`src/preload.ts`**: Renderer 프로세스에 안전한 IPC 브릿지를 노출.
- **`src/renderer/renderer.ts`**: UI 이벤트 처리 및 사용자 인터랙션 관리.
- **`src/naverBlogAutomation.ts`**: Puppeteer를 사용한 네이버 블로그 자동화 핵심 로직.
- **`src/contentGenerator.ts`**: AI 엔진을 사용한 구조화 콘텐츠 생성.
- **`src/imageGenerator.ts`**: DALL·E 3를 사용한 이미지 생성.
- **`src/imageLibrary.ts`**: Pexels API를 사용한 이미지 수집 및 관리.
