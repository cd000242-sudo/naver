사이트 편집

메인 문구, 제품 상세, 요금제, 다운로드 링크를 저장하면 실제 사이트가 같은 데이터를 읽습니다.

홈 첫 화면

메인 페이지 Hero 영역과 공지 문구입니다. 줄바꿈은 그대로 사이트에 반영됩니다.

한 줄에 한 장씩 입력합니다. 기본값은 압축된 성과 이미지 11장 전체입니다. 형식: 이미지경로 | 제목 | 설명 | 지표 | 이미지 설명

구매 탭 상단 문구

요금제 카드

카드 금액은 VAT 포함 금액입니다. 기능 목록은 한 줄에 하나씩 입력합니다.

제품정보 페이지 문구

제품별 카드/상세/이미지

이미지는 `/images/...` 또는 외부 URL을 넣으면 됩니다. 지표는 `100건 | 일 최대 자동 발행` 형식입니다.

제품정보 페이지 하단 이미지/표

다운로드 페이지 문구

제품별 다운로드 카드

제품명, 버전 문구, 이미지, 플랫폼별 다운로드 버튼 문구와 URL을 직접 수정합니다.

사이트 콘텐츠 관리

제품정보, 무료 챗봇, 구매 문구, 신규 제품 초안을 서버 저장 콘텐츠로 관리합니다.

관리자 AI 작업자 설정

Codex와 Claude Code는 관리자 전용 서버 작업자입니다. 일반 사용자는 볼 수 없고, 관리자 작업 요청에만 선택값을 사용합니다.
