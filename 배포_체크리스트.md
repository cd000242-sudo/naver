## 📦 배포 준비 사항

### 1. 빌드 확인
```bash
npm run build
```
✅ **결과:** 빌드 성공 (오류 없음)

### 2. 배포 패키지 생성
```bash
# Windows 전용 배포 (추천)
npm run dist:win

# 또는 모든 플랫폼
npm run dist:all
```

### 3. 배포 파일 위치
- **경로:** `release/` 폴더
- **파일 종류:**
  - `Better Life Naver Setup x.x.x.exe` (설치 파일)
  - `Better Life Naver x.x.x.exe` (포터블 버전)
  - `win-unpacked/` (압축 해제 버전)

## 🔍 다른 PC에서 사용 가능 여부

### ✅ 완전히 가능합니다!

#### 방법 1: 설치 파일 배포 (추천)
1. `Better Life Naver Setup x.x.x.exe` 파일을 구매자에게 전달
2. 구매자가 설치 파일 실행
3. 설치 후 API 키만 입력하면 바로 사용 가능

#### 방법 2: 포터블 버전 배포
1. `Better Life Naver x.x.x.exe` 파일을 구매자에게 전달
2. 별도 설치 없이 바로 실행 가능
3. API 키만 입력하면 바로 사용 가능

#### 방법 3: 압축 해제 버전 배포
1. `win-unpacked/` 폴더 전체를 압축 (ZIP 또는 7z)
2. 구매자가 압축 해제 후 `Better Life Naver.exe` 실행
3. API 키만 입력하면 바로 사용 가능

## 🛡️ 배포 시 주의사항

### 1. 설정 파일 제거
✅ **자동 처리됨:** `scripts/reset-config-for-pack.js`가 자동으로 개인 설정 제거
- `dist/settings.json` 제외됨
- API 키, 네이버 계정 정보 등 개인 정보 자동 제거

### 2. 필수 파일 포함 확인
✅ **자동 포함됨:**
- `dist/**/*` (빌드된 파일)
- `node_modules/**/*` (의존성)
- `puppeteer` 관련 파일 (asar unpack)
- `assets/` (아이콘 등)

### 3. 라이선스 관리
- 현재 라이선스 시스템이 구현되어 있음
- 구매자별로 라이선스 키 발급 필요
- 라이선스 키 없이도 기본 기능은 사용 가능 (제한 있을 수 있음)

## 📝 구매자에게 전달할 사항

### 1. 설치 가이드
```
1. Better Life Naver Setup x.x.x.exe 실행
2. 설치 경로 선택 (기본값 권장)
3. 설치 완료 후 프로그램 실행
4. 환경 설정 → API 키 발급 및 입력
5. 메인 화면에서 네이버 계정 입력
6. 바로 사용 시작!
```

### 2. 필수 API 키 안내
```
✅ 무료 API (필수):
- Gemini API (Google AI Studio)
- Pexels API (Pexels 웹사이트)

⚠️ 유료 API (선택):
- OpenAI API
- Claude API
```

### 3. 시스템 요구사항
```
- OS: Windows 10 이상 (64bit)
- RAM: 4GB 이상 (8GB 권장)
- 디스크: 500MB 이상 여유 공간
- 인터넷: 필수 (API 호출용)
```
