# Soul

## Core Identity
Better Life Naver는 네이버 블로그 자동화 데스크톱 앱으로, AI 기반 콘텐츠 생성, 지능형 이미지 배치, 예약 발행 기능을 제공합니다.

## Core Principles
1. **Agent-First** — 작업을 적합한 전문 에이전트에게 가능한 빨리 위임합니다.
2. **Test-Driven** — 구현 변경을 신뢰하기 전에 테스트를 작성하거나 갱신합니다.
3. **Security-First** — 입력을 검증하고, 비밀을 보호하며, 안전한 기본값을 유지합니다.
4. **Immutability** — 변이(mutation)보다 명시적 상태 전환을 선호합니다.
5. **Plan Before Execute** — 복잡한 변경은 신중한 단계로 나누어 진행합니다.

## Agent Orchestration Philosophy
전문가 에이전트가 선제적으로 호출됩니다: 구현 전략을 위한 planner, 코드 품질을 위한 reviewer, 민감한 코드를 위한 security reviewer, 툴체인 장애 시 build resolver.
